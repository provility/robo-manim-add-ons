# Arrow Utilities Demos

Screenshots and videos from ArrowUtil examples showcasing advanced arrow capabilities.

## Demos (arrow_utils_demo.py)

- **BasicArrowDemo** - Simple arrow with textbook-style tips
- **DashedArrowDemo** - Dashed arrow lines
- **PerpendicularBufferDemo** - Arrows with perpendicular offset
- **BidirectionalArrowDemo** - Arrows with tips on both ends
- **CurvedArrowDemo** - Arrows along circular arcs
- **LabelPositioningDemo** - Automatic label positioning with perpendicular offset
- **MarkerDemo** - Directional markers on paths
- **CombinedFeaturesDemo** - Multiple features combined (dashed + labels)
