# Geometry Demos

Screenshots from geometry utility examples demonstrating `perp()` and `parallel()` functions.

## Static Demos (geometry_demo.py)

- **PerpDemo** - Basic perpendicular line construction
- **ParallelDemo** - Basic parallel line construction
- **PlacementDemo** - Different placement options (start, mid, end)
- **GeometryComboDemo** - Combining perpendicular and parallel lines

## Dynamic Demos (dynamic_geometry_demo.py)

- **DynamicPerpExample** - Perpendicular line updating with rotation
- **DynamicParallelExample** - Parallel line following reference line
- **InteractivePerpUpdater** - Perpendicular following a moving point
- **MultiplePerpendicularUpdaters** - Multiple perpendiculars at different positions
- **ParallelLinesGrid** - Grid of parallel lines with dynamic updates
- **ParallelogramUpdater** - Dynamic parallelogram construction
