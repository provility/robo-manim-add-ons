# Annotation Demos

Screenshots from distance marker annotation examples using `distance_marker()`.

## Demos (annotation_demo.py)

- **BasicDistanceMarker** - Simple distance marker between two points
- **DistanceMarkerCustomization** - Different stroke widths (thin, normal, thick)
- **DistanceMarkerOrientations** - Horizontal and vertical orientations
- **DistanceMarkerRectangle** - Annotating rectangle dimensions
- **DistanceMarkerWithDots** - Markers with endpoint dots
- **LabelOffsetComparison** - Comparing different label offset values
- **MarkerOffsetComparison** - Comparing different marker offset values
- **MarkerOffsetExample** - Basic marker offset demonstration
- **MarkerOffsetPractical** - Practical use case with rectangle dimensions
- **PythagoreanTheorem** - Complete example: annotated right triangle with theorem
