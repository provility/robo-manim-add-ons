# Label Demos

Screenshots from vertex and edge label examples using `vertex_labels()` and `edge_labels()`.

## Demos (dynamic_labels_demo.py)

- **DynamicVertexLabelsExample** - Vertex labels following transforming triangle
- **DynamicEdgeLabelsExample** - Edge labels on transforming square
- **DynamicVertexAndEdgeLabels** - Both label types on pentagon
- **MorphingPolygonLabels** - Labels following morphing polygon
- **InteractivePolygonWithUpdater** - Labels showing real-time edge lengths
- **AnimatedLabelColors** - Dynamic color interpolation for labels
- **MultiplePolygonsWithLabels** - Multiple polygons with independent labels
