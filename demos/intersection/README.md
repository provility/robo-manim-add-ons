# Intersection Demos

Screenshots from intersection examples using `intersect_lines()` and `intersect_line_circle()`.

## Line-Line Intersection Demos

- **BasicIntersectionDemo** - Simple perpendicular line intersection
- **DiagonalIntersectionDemo** - Diagonal lines forming an X
- **ParallelLinesDemo** - Parallel lines (no intersection)
- **ExtendedIntersectionDemo** - Lines extended infinitely to intersect
- **DynamicIntersectionDemo** - Intersection following rotating line
- **MultipleIntersectionsDemo** - Multiple lines intersecting one line
- **TriangleIntersectionDemo** - Finding triangle orthocenter
- **ConditionalIntersectionDemo** - Checking if intersection exists

## Line-Circle Intersection Demos

- **BasicLineCircleIntersection** - Line through circle center (2 points)
- **TangentLineDemo** - Tangent line to circle (1 point)
- **NoIntersectionDemo** - Line missing circle (0 points)
- **DynamicLineCircleIntersection** - Intersections following rotating line
- **MultipleCirclesIntersection** - Line through multiple circles
- **ChordLengthDemo** - Measuring chord length from intersections
- **DiameterDemo** - Comparing diameter vs chord lengths
