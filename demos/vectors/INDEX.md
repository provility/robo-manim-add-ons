# Vector Operations

Demonstrations of vector operations using `VectorUtils`.

---

## BackwardVectorDemo
**Copy and shift a vector backward**

Create a copy of a vector shifted backward by a distance.

---

## ForwardVectorDemo
**Copy and shift a vector forward**

Create a copy of a vector shifted forward by a distance.

---

## VectorAdditionDemo
**Vector addition using tip-to-tail method**

Shows tip-to-tail vector addition with the resultant vector.

---

## VectorSubtractionDemo
**Basic vector subtraction: a - b = a + (-b)**

Demonstrates vector subtraction by reversing and adding vectors.

---

## DynamicVectorSubtraction
**Interactive subtraction that updates as vectors change**

Dynamic visualization where subtraction updates as input vectors are transformed.

---

## DotProductProjectionDemo
**Vector projection with step-by-step visualization**

Shows angle between vectors, projection line with right angle, and resulting projection vector.

---

## DynamicProjection
**Interactive projection that updates with vector rotation**

Projection automatically updates as vectors rotate, showing how the projection changes dynamically.

---

##BasicDecompositionDemo
**Vector decomposition into parallel and perpendicular components**

Decomposes a vector into components parallel and perpendicular to a reference vector.

---

## RotatingVectorDecomposition
**Decomposition updates as vector rotates**

Dynamic decomposition that updates as the vector being decomposed rotates around the origin.

---

## RotatingReferenceDecomposition
**Decomposition updates as reference vector rotates**

Dynamic decomposition that updates as the reference vector rotates.

---

## ScalingVectorDecomposition
**Decomposition with vector magnitude changes**

Shows how decomposition components scale proportionally as the vector magnitude changes.

---

## InclinedPlaneForceDecomposition
**Practical example: Force decomposition on inclined plane**

Physics application showing gravity force decomposed into components parallel and perpendicular to an inclined plane.

---

## MultipleDecompositions
**Multiple simultaneous decompositions with updaters**

Multiple vectors decomposed relative to one reference vector, all updating dynamically.

---

## OppositeDirectionDecomposition
**Decomposition when vectors point in opposite directions**

Shows how the parallel component can be negative when vectors point in opposite directions.

---

## ReverseAtDemo
**Vector reversal for subtraction visualization**

Demonstrates creating a reversed copy of a vector at a specified point.

---

## VectorParallelogramDemo
**Vector addition using parallelogram method**

Shows vector addition using the parallelogram law.

---

## MultipleVectorAdditionDemo
**Adding multiple vectors in sequence**

Demonstrates adding three or more vectors using tip-to-tail method.

---

## DynamicVectorAdditionDemo
**Interactive vector addition with dynamic updates**

Addition updates dynamically as input vectors change.

---

## ShiftAmountVectorAdditionDemo
**Using shift_amount for vector addition**

Alternative approach to vector addition using the shift_amount utility.

---

## CombinedVectorOperationsDemo
**Multiple operations in one scene**

Shows addition, subtraction, and scalar multiplication together.
