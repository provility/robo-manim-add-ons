# Vector Demos

Screenshots and videos from vector utility examples demonstrating VectorUtils methods for vector operations.

## Vector Reversal (vector_subtraction_demo.py)

- **ReverseAtDemo** - Vector reversal for subtraction visualization

## Vector Projection (dynamic_geometry_demo.py, vector_projection_demo.py)

- **ProjectionDemo** - Vector projection with dynamic rotation
- **ProjectionScalingDemo** - Projection scaling with vector magnitude
- **DotProductProjectionDemo** - Vector projection with step-by-step visualization
- **ProjectionWithFormula** - Projection with mathematical formula display
- **DynamicProjection** - Interactive projection that updates with vector rotation
- **ProjectionWithRegion** - Projection with shaded triangular region
- **ProjectionCases** - Different projection cases: acute, right, and obtuse angles
- **ProjectionScaling** - Projection magnitude changes with vector magnitude
- **DotProductVisualization** - Visualize dot product as projection times magnitude

## Vector Decomposition (vector_decomposition_demo.py, vector_projection_demo.py)

- **BasicDecompositionDemo** - Vector decomposition into parallel and perpendicular components
- **RotatingVectorDecomposition** - Decomposition updates as vector rotates
- **RotatingReferenceDecomposition** - Decomposition updates as reference vector rotates
- **ScalingVectorDecomposition** - Decomposition with vector magnitude changes
- **InclinedPlaneForceDecomposition** - Practical physics example with inclined plane
- **MultipleDecompositions** - Multiple simultaneous decompositions with updaters
- **OppositeDirectionDecomposition** - Decomposition when vectors point in opposite directions
- **ProjectionDecomposition** - Decompose vector into parallel and perpendicular components
